## 涉及模块

| 序号 | 模块 | 主要功能 |
|------|------|----------|
| 1 | 认证与用户 | 登录、用户信息、主题偏好 |
| 2 | 分类管理 | 树形分类 CRUD（管理员） |
| 3 | 标准文件 | 搜索/列表/预览/下载/上传/编辑/删除/批量/导出 |
| 4 | OCR 与核验 | OCR 重试、人工核验 |
| 5 | 回收站 | 回收站列表/批量还原/彻底删除 |
| 6 | 审计日志 | 操作日志查询（管理员） |
| 7 | 系统公告 | 首页公告展示 |
| 8 | 系统设置 | 站点名称、注册开关（管理员） |

## 关键设计决策摘要

| 维度 | 决策 |
|------|------|
| 风格 | 现代简约 Flat Design |
| 色调 | 蓝色系，专业沉稳 |
| 主题 | 浅色 + 深色，跟随系统切换 |
| 字体 | 中文：霞鹜文楷；英文：Inter/Roboto；等宽字体用于标准号 |
| 布局 | 左侧菜单 + 顶部导航 + 内容区，菜单可折叠 |
| 列表 | 表格形式，管理员可扩展列 |
| 预览 | 页面内嵌 PDF.js |
| 管理 | 独立管理后台入口，仪表盘概览 |
| 终端 | 仅桌面端 |
| 语言 | 仅中文 |
