## 四、首页 / 工作台

### 4.1 管理员仪表盘

2. 是否需要高级搜索/筛选面板？

   - [x] 是

### 4.3 标准文件列表

1. 列表展示形式：

2. 默认展示列：
   - [x] 标准号 `standard_no`

   - [x] 发布日期 `issue_date`
   - [x] 实施日期 `implement_date`
3. 是否需要用户自定义列显示？
   - [x] 是
4. **补充**：表格列后期可由管理员增加
5. 状态字段（OCR/核验）的展示方式：
   - [x] 彩色标签 Tag（如绿色=通过，黄色=待处理，红色=失败）

   - [x] 10

### 4.4 系统公告
